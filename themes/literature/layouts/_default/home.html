Hi. This site mostly exists for my own reference notes and to practice my technical writing from time to time. Occassionally, I will show off notable software projects that I'm able to freely share. My particular interests revolve around Windows malware, assembly, and at one point in time, meteorology.

The site design is influenced heavily by gwern.net and, by extension, the "old" world of words-on-paper and plain hypertext. My posts leverage some dynamic content, but the site is still designed such that not much will change if that ever breaks.

The repo backing this site contains some utilities for archiving references and generating citations. The static site tool in use is Hugo with some shortcode extensions for making things a bit fancier, including bibliographies, footnotes, figures, and quotes.

Enjoy!
